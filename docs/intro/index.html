Research Engineering Learning Path

A Beginner-Friendly Guide for Citizen Researchers at Average Joes Lab

📚 This guide has been split into interconnected modules for better learning progression. Choose your starting point below or follow the recommended path.

📋 Learning Modules​

This comprehensive guide is organized into 9 interconnected modules. New to research? Follow them in order. Have some experience? Jump to relevant sections.

🚀 Quick Access​

ModuleFocusTimeFor You If...
01. IntroductionWhat is research engineering?10 minYou're completely new
02. Quick StartGet immediate wins30 minYou want to try it now
03. Setup & RequirementsHardware, costs, setup1-2 hoursYou need to know what's required
04. PrerequisitesEssential skills2-8 weeksYou need to learn basics
05. Learning Framework4-phase progression16 weeksYou want the full program
06. Tools & ResourcesSoftware and platformsReferenceYou need tool guidance
07. Action PlanStep-by-step checklistOngoingYou want clear next steps
08. TroubleshootingCommon issues & solutionsAs neededYou're stuck
09. CommunitySupport and collaborationOngoingYou want to connect

🎯 Choose Your Path​

🆕 Complete Beginner​

"I've never coded or read a research paper"

  1. Start with 01. Introduction to understand what research engineering is
  2. Do the 02. Quick Start for immediate confidence
  3. Review 03. Setup & Requirements to understand what you need
  4. Work through 04. Prerequisites at your own pace
  5. Follow the 05. Learning Framework for structured progression

🔧 Some Technical Background​

"I can code but I'm new to research"

  1. Read 01. Introduction for context
  2. Skim 04. Prerequisites to identify gaps
  3. Jump to 05. Learning Framework Phase 1 or 2
  4. Use 06. Tools & Resources as needed

🎓 Research Experience​

"I know research but want to learn engineering"

  1. Review 01. Introduction for our approach
  2. Check 06. Tools & Resources for technical tools
  3. Start with 05. Learning Framework Phase 2 or 3
  4. Engage with 09. Community to share your expertise

🆘 Stuck or Confused​

"I need help or have questions"

  1. Check 08. Troubleshooting for common issues
  2. Join 09. Community for support
  3. Use 07. Action Plan to get back on track

💡 Key Principles​

What Makes This Different​

  • 🎯 Realistic expectations - We're honest about what beginners can accomplish
  • 🛠️ Practical focus - Learn by doing, not just reading
  • 🤝 Community-driven - Learn with and from others
  • 📈 Progressive difficulty - Build skills step by step
  • 🆓 Accessible - Everything essential is free

Our Approach to Papers​

We carefully select papers and explain exactly what you should do:

Paper TypeWhat You'll DoWhy This Works
🟢 Train from scratchSmall datasets, quick trainingBuild understanding from ground up
🟡 Fine-tuneUse pretrained model, adapt to new taskLearn practical ML workflows
🔵 Run inferenceUse existing model to generate outputsUnderstand model capabilities

🚀 Ready to Start?​

Recommended Starting Points​

🆕 Brand New? → 02. Quick Start (30 minutes for immediate wins)

🎯 Want the Full Program? → 01. Introduction (comprehensive overview)

🔧 Need Technical Setup? → 03. Setup & Requirements (hardware and costs)

🤝 Want Community? → 09. Community (join Discord and connect)

💡 Remember: Every expert was once a beginner. This modular approach lets you learn at your own pace while staying connected to a supportive community of citizen researchers.
