03. Setup & Requirements

Everything you need to know about hardware, costs, and technical setup

💻 Hardware & Cost Requirements​

Minimum Requirements (Free Tier) 💰​

ComponentRequirementNotes
ComputerAny laptop/desktop with internetEven old machines work!
RAM4GB minimum (8GB recommended)For local development
Storage5GB free spaceFor downloads and projects
InternetStable connectionFor cloud computing

Cloud Computing (Recommended - FREE) ☁️​

PlatformWhat You GetLimits
Google ColabFree GPU access12-hour sessions
Kaggle NotebooksFree GPU/TPU30 hours/week
Hugging Face SpacesFree model hostingCommunity tier

Cost Breakdown 💸​

PhaseCostWhat's Included
Phase 1-2$0Everything free
Phase 3-4$0-20/monthOptional Colab Pro for longer sessions
Total (16 weeks)$0-80Completely optional

What You DON'T Need ❌​

  • 🚫 Expensive GPU ($1000+ graphics cards)
  • 🚫 Powerful desktop computer
  • 🚫 University access or credentials
  • 🚫 Paid software licenses

💡 Bottom line: You can complete this entire learning path for $0!

🔧 Browser Setup​

Recommended Browsers​

  • Chrome (best Colab compatibility)
  • Firefox (good alternative)
  • Safari (works but some limitations)

Browser Extensions (Optional)​

  • GitHub Desktop - For easier Git management
  • Grammarly - For writing papers and documentation

🚀 Ready for Prerequisites?​

Once you've completed the required setup checklist above, you're ready to move on to 04. Prerequisites.

🆘 Setup Issues?​

Common Problems & Solutions​

ProblemSolution
"Can't access Colab"Try incognito/private browsing mode
"GitHub signup blocked"Use different email or try later
"Discord won't load"Check firewall/network restrictions
"Don't have Google account"Any email provider works for GitHub/Discord
